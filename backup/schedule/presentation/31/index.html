Taller de Python. Desde nada, hasta el infinito y mas alla!

Ariel Silvio

Audience level:
Principiante

Descripción

Un Taller donde cualquier persona con la inquietud de saber, ver y aprehender Python pueda hacerlo. Dar una idea sencilla, robusta y acertada de programación y un lenguaje de programación practico, escalable como Python, sus modulos o librerías y aplicaciones en las diferentes areas hoy en dia. Llegar a usar Python en una Raspberry Pi. Acompaño con Mapas conceptuales y palabras disparadores.

Resumen

Sentarnos en un banquito o mesa con sus respectivas maquinas (notebooks o Netbooks) y empezar buscando en la web lo que es python y un lenguaje de programacion. Obtener los instaladores del interprete o bien pasarselo desde un pendrive. Buscar los materiales de consulta como Videos, Libros y Tutoriales en PDF o HTML y sitios de referencia para la formacion y consulta, como StackOverflow y Mouse vs Python. Ver materiales generados y editados de manera personal por diferentes programadores de Python, locales y del extranjero. Saber interpretar los errores y buscar la solucion en la web. Ser la guia desde el principio como muchos hubieramos querido tener cuando empezamos con Python. Enseñar lo manera conveniente de encarar la programacion para que ellos no tengan problemas que uno paso. Transmitir la esencia del Software Libre como comunidad, esta vez expresado de la mejor manera en este lenguaje de programacion. Sumarlos a la comunidad Python. Usar el metodo de DESCARTES: desde lo sencillo a lo complejo. Comentar experiencias extras como el uso de NVDA para No Videntes. NVDA esta hecho en Python y es codigo abierto y funciona en Windows.
