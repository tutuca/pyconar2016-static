Taller de programación para mujeres Django Girls, Bahía Blanca.

Argentina en Python es un proyecto personal y comunitario que promueve el uso del lenguaje de programación Python para resolver problemas cotidianos en usuarios comunes, como así también desarrollar programas poderosos y complejos de una forma sencilla, estimulando el aprendizaje colaborativo y la filosofía del Software Libre.

Desde Argentina en Python queremos animar a más mujeres a dominar la Web, es por eso que estamos organizando este taller<br/> en el cual podrás construir tu propio sitio web desde cero en un día. Además, vamos a utilizar todas las herramientas<br/> que programadores profesionales utilizan diariamente. ¡Y de una manera muy sencilla!

Charlas/Talleres

Taller de Programación para Mujeres Django Girls

Viernes 08:00–19:00 in Salón "Fundadores"
